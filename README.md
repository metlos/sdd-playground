# SDD playground

A simple repository with a bootstrap command and couple of skills to help
with spec-driven development (SDD) using speckit.

Clone and start by running `/project-bootstrap` in either Claude Code or Cursor
(tested in Claude but Cursor should work, too).

Apart from the project bootstrap, it is necessary to install and configure
a Jira MCP server. Follow the `/initialize-jira` skill to set that up.
